el-cascader级联选择器供选择的列表太长

万事屋的打工仔 2022-06-15 前端dev-problem

element-ui在2.13.0版本中添加了el-cascader级联 (opens new window)选择器多选功能,但是在显示数据列表时太长了,超过了浏览器高度,导致列表上面的数据看不到

在我们的全局global.css中添加

.el-cascader-menu {
height: 200px;
}
